# 📦 Exercícios de Programação Orientada a Objetos (POO) — Java

Este repositório contém os exercícios que estou desenvolvendo durante meus estudos de **Programação Orientada a Objetos em Java**, seguindo o curso do professor Nélio Alves.

O objetivo deste repositório é **praticar os conceitos fundamentais da POO**, consolidando a teoria através da implementação de exercícios propostos ao longo do curso.

---

# 🎯 Objetivos do Repositório

- Praticar **Programação Orientada a Objetos**
- Desenvolver **lógica de programação**
- Aprender a estruturar projetos em **Java**
- Evoluir na escrita de código limpo e organizado
- Criar um histórico público da minha evolução como desenvolvedor

---

# 📚 Conteúdos Praticados

Durante os exercícios, são trabalhados conceitos como:

- Classes e Objetos  
- Atributos  
- Métodos  
- Encapsulamento  
- Construtores  
- Sobrecarga de métodos  
- Modificadores de acesso  
- Arrays e listas  
- Estrutura de repetição e decisão aplicadas à POO
